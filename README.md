JarLoc - AI Mod Translator for Minecraft

**JarLoc** is a free, open-source web tool Designed to automate the tedious process of translating Minecraft mods.

It uses advanced AI models (Google Gemini) to translate language files (`.json`) directly within your browser and automatically generates a ready-to-use **Resource Pack** with the translations.

**Try it online:** [https://jarloc-app.vercel.app/](https://jarloc-app.vercel.app/)

---

##  Key Features

- No Installation Required:** Runs entirely in your web browser.
- Client-Side Security:** Your API Key is sent directly from your browser to Google's servers. It never passes through or is stored on JarLoc's servers. The code is 100% open source for auditing.
- AI-Powered Translations:** Uses Google's Gemini models for context-aware translations.
- Auto-Repair System:** Smart algorithms detect and fix broken JSON syntax often generated by AI in large files.
- Instant Resource Pack:** Generates a compliant `.zip` file ready to be dropped into your `resourcepacks` folder. No manual file editing needed.

---

##  How to Use

1.  **Get a free API Key:** Go to [Google AI Studio](https://makersuite.google.com/app/apikey) and create a free API key (the free tier is very generous).
2.  **Upload your Mod:** Drag and drop the `.jar` or `.zip` file of the mod you want to translate.
3.  **Select File & Model:** Choose the language file (e.g., `en_us.json`) from the list. Paste your API Key and select an AI Model.
    * *Tip: For very large mods, use "Pro" models instead of "Flash/Lite" models to avoid incomplete responses.*
4.  **Translate & Download:** Click "Traducir con IA". Once finished, review the result and click "Descargar ZIP".
